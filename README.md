## Usage

`\H3::render($layout)` renders *layout* .html by F3 Template
 
`\H3::gen($length)` returns random generated key
 
`\H3::clean($string)` returns *slug*
 
`\H3::dump($var)` echos $var in <pre>

## Installation

Via composer: `"resist/H3": "dev-master"`
